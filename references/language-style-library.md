# Language Style Library

用于补足不同汇报场景下的中文咨询表达、标题链和负面判断替代表达。写标题、摘要、建议、风险和会议纪要口径时读取本文件，并与 `consulting-language-playbook.md` 同时使用。

## Audience Tone Profiles

| Audience | Tone | Prefer | Avoid |
|---|---|---|---|
| 董事会 | 稳健、决策导向、保留前提 | `建议授权`、`需设置边界`、`应优先验证` | 过度技术细节、无条件判断 |
| 管理层 | 行动导向、责任清楚 | `由...牵头`、`以...为监控指标`、`按季度复盘` | 只写方向、不写动作 |
| 投委会 | 风险收益、情景和退出 | `基准/压力情景`、`下行保护`、`触发条件` | 单点预测、无敏感性 |
| 监管/合规 | 依据、审慎、口径边界 | `在当前披露口径下`、`需结合监管要求确认` | 强价值判断、口号式建议 |
| 培训分享 | 机制、案例、可复制做法 | `该案例提示`、`可复制环节`、`适用边界` | 过度结论化、缺案例事实 |
| 投资人/资本市场 | 估值、预期、可验证指标 | `估值修复需验证`、`市场定价可能关注` | 把市场反应写成必然 |

## Negative Judgment Replacements

| Weak / risky | Consulting-ready |
|---|---|
| `表现较差` | `低于样本中位 X bp / X pct，处于样本后段` |
| `风险很大` | `风险暴露已具备可识别特征，但损失水平仍需量化` |
| `不可持续` | `当前证据尚不足以支持持续改善判断` |
| `管理薄弱` | `指标波动显示该环节需要补充监控阈值和责任边界` |
| `估值被低估` | `PB 水平低于可比样本中位，估值修复仍需经营指标验证` |
| `没有竞争力` | `相较同业中位，核心指标存在 X 项缺口` |
| `必须整改` | `建议优先纳入整改清单，并设置完成时点和验收指标` |
| `严重依赖投资收益` | `利润改善中投资端贡献占比较高，主业贡献仍需单独验证` |

## Judgment Pattern Library

当不确定如何把判断写得专业、稳健、可交付时，按判断类型选择以下句式。所有句式都应替换为真实指标、样本和来源，不得空填。

### A. 指标好于同业

- `[主体] 在 [指标] 上高于样本中位 [X]，但是否形成经营优势仍需结合 [穿透指标] 判断。`
- `从 [样本/期间] 看，[指标] 位于样本前段，说明 [能力/结果] 已具备相对支撑。`
- `[指标] 领先更多体现为 [表层结果]，后续应验证其是否转化为 [客户关心结果]。`

### B. 指标差于同业

- `[指标] 低于样本中位 [X]，短板主要影响 [经营/风险/估值/现金流] 判断。`
- `[主体] 在 [指标] 上处于样本后段，提示 [根因] 需要进一步拆解。`
- `该差距不宜直接写成能力不足，更应表述为 [指标口径] 下的相对缺口。`

### C. 趋势向好

- `[指标] 连续 [期间] 改善，初步显示 [管理动作/市场环境] 对 [结果] 形成支撑。`
- `改善趋势已经出现，但持续性仍需通过 [后续指标] 交叉验证。`
- `[指标] 从 [A] 提升至 [B]，对 [客户决策] 的含义是 [管理含义]。`

### D. 趋势恶化

- `[指标] 连续 [期间] 下行，压力主要集中在 [驱动因素]。`
- `趋势变化提示 [风险/成本/收入] 端压力上升，需要在 [时间窗口] 内验证是否可逆。`
- `[指标] 的边际变化已超过 [阈值/同业中位]，应纳入后续复盘指标。`

### E. 指标之间存在矛盾

- `[表层指标] 改善，但 [穿透指标] 未同步改善，说明 [表层结果] 尚未完全转化为 [价值/质量/可持续性]。`
- `[经营视角] 与 [市场/实证视角] 指向不完全一致，需通过 [调和解释/补充指标] 验证。`
- `[短期支撑] 可以解释当前结果，但 [长期约束] 仍限制后续改善空间。`

### F. 建议采取行动

- `建议由 [责任主体] 在 [时间] 前完成 [动作]，并以 [指标/阈值] 作为验收标准。`
- `建议先验证 [关键假设]，再决定是否推进 [资源投入/授权事项/交易安排]。`
- `在 [前提] 成立时优先推进 [路径A]；若 [约束] 未解除，则保留 [路径B] 作为过渡。`

### G. 不行动后果

- `若不调整 [动作/机制]，[风险/缺口] 可能继续传导至 [客户关心结果]。`
- `若维持现有路径，后续判断仍将受 [关键不确定性] 制约。`
- `不行动的主要代价不是短期数字波动，而是 [能力/风险/资本/估值] 约束持续存在。`

### H. 限制条件 / 不确定性

- `上述判断基于当前披露口径，仍需结合 [缺失数据/后续披露/管理层口径] 验证。`
- `由于 [样本/期间/口径] 存在差异，排名结论仅在当前可比口径下成立。`
- `该测算为 RSM 情景假设，不构成预测；实际结果取决于 [关键变量]。`

## Dual-Perspective Tone Rules

双视角页面应体现独立证据来源和语气差异，不要把同一段话换标签复制。

- **经营视角**：使用归因分析语气，强调来源、驱动和管理动作。
  - `利润改善的主要贡献来自 [驱动项]，但 [质量指标] 仍需单独验证。`
  - `[指标变化] 更可能由 [经营驱动] 解释，后续应纳入 [复盘机制]。`
- **市场/实证视角**：使用统计发现语气，强调相关性、显著性、定价或外部验证。
  - `回归/样本对比显示，[指标] 与 [估值/风险折价] 存在方向一致的关联。`
  - `市场定价尚未充分反映 [经营指标]，说明估值修复仍需可验证信号。`
- **风险/合规视角**：使用审慎边界语气，强调前提、限制和触发条件。
  - `在当前证据下，[风险] 已具备可识别特征，但损失水平仍需量化。`
  - `建议先完成 [证据/口径/条款] 复核，再判断是否进入下一阶段。`

## Before / After Rewrite Bank

用于客户正式材料、董事会/管理层汇报和语言润色 pass。改写时保留事实边界，不把弱证据写成强结论。

| Weak draft | Consulting-ready rewrite | Why it works |
|---|---|---|
| `整体表现较好，但仍需关注风险。` | `利润端改善已出现，但资产质量迁徙和费用效率仍需纳入季度复盘，避免只用总量利润判断经营质量。` | 从空泛评价改为发现、风险点和管理动作 |
| `建议持续优化资产质量管理。` | `建议将关注类迁徙率、逾期 90 天以上贷款和不良生成率设为三项月度监控指标，连续两个季度偏离阈值时启动专项复盘。` | 补对象、指标、阈值和动作 |
| `公司估值偏低，未来有修复空间。` | `PB 水平低于可比样本中位，但估值修复仍需核心营收增速、风险调整收益和分红稳定性共同验证。` | 避免直接承诺市场反应，加入验证条件 |
| `投资收益是主要原因。` | `利润改善中投资端贡献占比较高，但是否可持续仍需结合权益市场表现、资产配置结构和承保贡献变化判断。` | 把归因和可持续性边界分开 |
| `各国政策差异较大。` | `各司法辖区差异不只是核销时间不同，而是监管哲学、税务协同和资本约束组合不同。` | 把罗列式事实提升为分析框架 |
| `方案二更好。` | `在执行成本可控且数据口径可统一的前提下，优先推进方案二；若系统改造窗口不足，则保留方案一作为过渡路径。` | 说明前提、推荐路径和备选路径 |
| `风险较高，不建议推进。` | `当前证据显示下行风险尚未被充分缓释，建议先完成抵质押覆盖、现金流压力测试和合同触发条款复核，再决定是否进入下一阶段。` | 将否定判断转化为决策门槛 |
| `监管要求比较复杂。` | `监管要求涉及会计、税务和资本三类约束，建议先按适用主体和触发条件拆分，再判断是否具备制度借鉴空间。` | 给出拆解维度和下一步判断方式 |

## Meeting-Minutes Ready Sentences

客户会议或董事会材料中的标题、结论条和建议应能直接进入会议纪要：

- `会议确认，后续监控应从利润总量转向承保质量、费用效率和投资收益稳定性三条线。`
- `会议建议，项目组先完成口径统一和敏感性测算，再提交方案路径选择。`
- `会议同意，将关注类迁徙率和逾期 90 天以上贷款纳入季度经营复盘。`
- `会议要求，在正式采用外部政策经验前，补充适用主体、监管边界和税务协同条件。`

避免写成：

- `会议认为整体较好。`
- `会议要求持续关注。`
- `会议建议进一步优化。`

## Title Spine Examples

### 银行价值创造 / 苏农风格

```text
客户问题：苏农银行估值修复应优先验证哪些经营指标？
客户答案：账面利润具备支撑，但估值修复更依赖核心营收、中收转化、资产质量和资本效率四类指标同步改善。

P01 账面利润改善尚未完全转化为资本市场定价修复
P02 VQA 框架将估值问题拆为账面收益、风险调整和资本市场三层
P03 ROA 位于样本前段，但核心营收增速低于同业中位
P04 息差压缩是核心营收承压的主要来源
P05 中收转化不足削弱了对息差下行的对冲能力
P06 个贷不良暴露使风险调整收益率仍需验证
P07 资本内生增速约束资产扩张和分红空间
P08 PB 折价与中收转化和广义不良缺口方向一致
P09 估值修复应优先围绕中收、息差、资产质量三类抓手推进
P10 未来 12 个月应以月度指标和季度复盘验证修复路径
```

### 全球政策比较

```text
客户问题：不同司法辖区贷款核销制度差异，对银行风险管理有什么启示？
客户答案：各辖区差异不只是核销时间不同，而是监管哲学、税务协同和资本约束的组合差异；借鉴时应区分市场出清、财政协同和审慎缓冲三类逻辑。

P01 八大司法辖区的核销制度体现三类监管哲学
P02 美国以强制核销推动市场清洁，但依赖成熟二级市场承接
P03 欧洲以资本惩罚约束长期挂账，银行行为更受资本成本驱动
P04 新加坡通过税务抵扣降低审慎处置的财务摩擦
P05 香港以监管储备保留资本缓冲，处置节奏更偏审慎
P06 日本长期处置经验显示政策稳定性优先于快速出清
P07 巴西改革将司法处置转向规则化和算法化治理
P08 对中国银行的启示是核销政策需同时设计会计、税务和资本联动
```

### 保险新准则财务结果

```text
客户问题：新准则下上市险企利润改善是否具备持续性？
客户答案：利润普遍改善，但增长来源分化明显；可持续性需要回到承保质量、费用效率和投资收益稳定性验证。

P01 2025 年样本险企利润普遍改善，但增长来源分化扩大
P02 投资端回暖是利润改善的重要贡献项，承保端贡献仍需拆分验证
P03 保险服务收入增长并不必然转化为承保利润改善
P04 费用效率差异决定利润改善能否沉淀为经营质量
P05 综合收益波动提示投资收益稳定性仍是后续观察重点
P06 后续监控应从利润总量转向承保、费用和投资三线指标
```

## Suggestion Sentence Patterns

建议必须包含对象、动作、指标、时间或触发条件：

- `建议由 [责任主体] 在 [时间] 前完成 [动作]，并以 [指标/阈值] 作为验收标准。`
- `建议将 [指标1/指标2/指标3] 纳入月度监控，连续 [周期] 偏离阈值时启动专项复盘。`
- `在 [前提] 成立时，优先推进 [路径A]；若 [约束] 未解除，则保留 [路径B] 作为备选。`
- `建议先验证 [关键假设]，再决定 [资源投入/方案选择/授权事项]。`

## Executive Summary Language

执行摘要每条建议使用：

```text
结论：一句直接答案。
证据：2-3 个已证明事实。
含义：对客户决策的影响。
建议：下一步动作、责任和指标。
```

禁止：

- `整体向好，但仍需关注`
- `建议持续加强`
- `未来仍有提升空间`

改为：

- `利润端改善已出现，但核心营收和风险调整收益尚未同步验证，建议将中收转化、息差对冲和资产质量迁徙设为三项季度复盘指标。`
