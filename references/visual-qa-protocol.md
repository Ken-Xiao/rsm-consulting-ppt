# Visual QA Protocol

用于生成 PPT 后的截图级视觉检查。复杂金融 PPT 不允许第一次导出后直接交付。

## Required Loop

1. 导出 PPTX。
2. 渲染为预览图或 contact sheet。
3. 逐页检查视觉问题。
4. 记录问题。
5. 修复受影响页面。
6. 重新渲染受影响页面。
7. 直到一轮检查没有 critical 问题。

## Checks

### Reference PDF Match

如果目标是对标 `2025年上市保险公司新准则财务结果分析_final.pdf` 或 `final-20251031 商业银行及保险公司不良处置行业实践分享-1103.pdf`，先读取 `references/rsm-reference-visual-dna.md`，逐项检查：

- visual profile 是否正确：保险数据页用 `rsm-insurance-results`，实践分享页用 `rsm-practice-sharing`。
- 保险正文页是否有固定页眉、RSM 三色短条、右上模块胶囊、浅灰结论条、白色阴影图表卡。
- 保险封面/章节页是否有照片区、白色几何装饰、右侧色条或编号块。
- 实践分享正文页是否有亮蓝书法标题、三段标题线、弱页脚。
- 实践分享封面/目录/章节页是否有建筑照片拼贴和白色几何方块。
- 章节页是否符合“章节编号 + 章节名称 + 专业图片 + 可选短章节问题”的结构；不得出现正文 bullet 或密集说明。
- 使用生成图片时，是否记录 prompt、生成工具和使用边界；图片不得出现真实 logo、可读文字或人物特写。
- 是否混用了两套视觉：例如保险正文页出现书法标题，或实践分享页出现保险模块胶囊。
- 图表色板是否固定为亮蓝、深蓝、灰、浅蓝；不要出现随机高饱和色。

### Layout

- 标题、正文、图表、页脚是否重叠。
- 文字是否被截断、溢出或贴边。
- 卡片、列、流程节点是否对齐。
- 页面边距是否过窄。
- 是否存在某处过空、某处过挤。

### Typography

- 标题层级是否清楚。
- 同类文字字号是否一致。
- 小字是否仍可读；正文页普通正文不得低于 18pt，密集矩阵不得低于 12pt。
- 数字和单位是否贴合。
- 若文字过多导致字号变小，应拆页、删减或改版式，而不是继续缩小。

### Typography Thresholds

- 正文页大标题：不低于 32pt。
- 卡片标题：不低于 20pt。
- 普通正文：不低于 18pt。
- 表格/矩阵正文：常规不低于 13pt；极密矩阵最低 12pt 且只能用于附录或横向大矩阵。
- 页脚/来源：不低于 10pt。
- 若 contact sheet 中无法读出每页主标题和卡片标题，视为 `critical`。

### Color And Contrast

- 浅色文字是否放在浅色背景上。
- 语义色是否一致：绿=可行/正向，金=条件/风险，红=禁止/负向，蓝=主路径。
- 图表颜色是否过多。

### Chart Readability

- 缩略图下是否能识别主结论。
- 每张核心图表是否具备 `chart_point_of_view`、基准、焦点系列、单位、样本、期间和来源。
- 图表标注是否只突出 1-2 个与结论相关的重点。
- 坐标、单位、中位线/阈值线是否明确。
- 标签是否靠近对应数据点。
- 图例是否必要；能直接标注时不要图例。

### Image Professionalism

- 每张图片是否有明确 `image_role`。
- AI 生成图片是否只用于封面、章节或无事实含义背景。
- 证据图片是否有来源、日期和脱敏说明。
- 图片是否符合当前 visual profile，避免暖色广告风、抽象插画、logo、可读文字和人物特写。
- 图片是否抢占主证据或降低正文可读性。

### Content Artifacts

- 是否有占位符、模板示例公司名、假数据。
- 是否残留 HTML 模板示例文字或 `{{placeholder}}`。
- 来源条是否缺失。
- 页码是否连续。
- 法律/财务结论是否被视觉强调成无条件判断。

## Severity

- `critical`：影响理解或事实正确性，必须修。
- `warning`：影响专业感或阅读效率，尽量修。
- `suggestion`：可优化但不阻塞交付。

## Report Format

```json
{
  "visual_qa": [
    {
      "page": "P12",
      "severity": "critical",
      "issue": "热力矩阵右侧标签被截断",
      "fix": "增加右边距或缩短标签"
    }
  ]
}
```
