Methodology — six commitments
  1. Every percentage cites a CRM stage ID (slide 02 lock-in).
  2. Waterfall reconciles to ±1 % of CRM truth — off triggers re-audit.
  3. Push / Pull / Kill bounded at ≤ 10 named deals.
  4. Forecast accuracy uses last 4 quarters + confidence band.
  5. Rep slides use small multiples — no public ranked bar.
  6. Lost-deal analysis ranks by ARR, not count.
